### How to Take Screenshots:

1. Run your app: `flutter run -d chrome`
2. Press `F12` in browser
3. Click device toolbar icon (phone icon)
4. Select device: iPhone 12 Pro or similar
5. Navigate to each screen
6. Take screenshots (browser screenshot tool or `Windows + Shift + S`)
7. Save in this folder

### After Adding Screenshots:

```bash
git add screenshots/
git commit -m "Add app screenshots"
git push origin main
```

Then your README will automatically display them!
